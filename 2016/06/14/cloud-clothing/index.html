服装流行趋势预测机制

  • clothing
  • cloud

相关研究

关于流行色预测

  • 直觉预测、流行色专家会议预测、流行色情报预测、流行色市场统计分析预测、流行色偏爱值预测法、流行色回归分析预测法、流行色灰色预测法
  • 针对服装面料、图案、廓形等其他服装流行趋势预测分支上文献相对较少
  • 相比色彩预测没有那么严谨，多取决于行业专家对新科技、新材料、新资讯的把握

传统的流行趋势预测机制

  1. 直观预测机制
  2. 统计预测机制
  3. 数学模型预测机制

大数据优势

  1. 可供使用的数据资源量的规模以及数据的更新速度
  2. 多元化、智能化的流行趋势预测的资料收集和分析

所以对于预测效率是能够大大超越传统预测机制

预测方法的现状

  • 预测的内容可以按照品类、服装要素和制作原料三个方面来划分
  • 以品类分类可包括牛仔、皮革、针织、西装、鞋履、配饰、衬衫等等
  • 以服装要素分类主要为款式、图案、色彩三个方面

基于云计算的服装流行趋势预测机制

在维克托所著的《大数据时代》中提到这样一个例子

  • H1N1 流感爆发几周前，谷歌公司的工程师在《自然》杂志上发表了一篇令公共卫生官员和计算机科学家都为之震惊的论文。文中解释了谷歌通过人们在网上的搜索记录来预测冬季流感的传播，并达到以州和地区为单位的精确度
  1. 时尚的图片信息
    时尚图片通常包括每一季度的品牌发布会图片、街拍图片、具有美感的代表性摄影作品等
  2. 交易数据
    热销商品、消费者的消费记录和浏览记录、商家的销售记录、产品描述数据、这些数据可用于产品的相似识别、消费者的消费行为分析、由此可以做出更好的大数据推荐系统
  3. 时下热门的流行语
    热门流行用语是大数据背景下产生的新事物，由于信息的传播量和传播速度，一个信息会偶然地成为大众津津乐道的事情并因此派生出流行用语
  4. 其他
    大致包含了有关于生活和时尚的各种信息，电影、新闻、音乐、演出、明星等等。只要能够提炼出相关信息的数据，都应该是大数据包容的数据资源。

数据量化的规则制定

  1. 电商平台的标题检索词数据统计
  2. 电商平台上“爆款”产品的消费者信息分析
  3. 各大服装品牌的销售数据分析
  4. 来源于秀场、街拍、灵感、生活方式等一系列图片信息的量化

其中第1、2类信息需要联合大型的电商平台，或者从大型电商平台那里购买数据库中的数据，通过对得到的数据进行建模并分析；
第3类则需要企业内部的商业信息，因为其中涉及到行业竞争中企业不愿意共享的商业机密，使这成为最难得到的一类信息；
第4类数据在某种程度上可能具有一定的局限性，但这一类信息因其深入性和广泛性，成为最关键、最具有量化的可行性和必要性的数据。

量化问题需要几个步骤

  1. （第一级分类）首先，对于可获得的数据类型包括秀场、街拍、灵感、生活方式等一系列图片信息进行分析；其属性大致分为四类：男装、女装、灵感类图片、事件类图片
  2. （第二级分类）由于所面对的数据的服装属性，为进一步描述一类数据，再次细化其属性为：色彩、款式、图案。其中图案又从图案类别和图案特征两个维度来描述

服装流行趋势预测平台的结构

  1. 数据的搜集环节
    数据的搜集环节主要涉及到的是数据字典的编写工作，也就是数据录入规则 的制定，数据的量化办法

  2. 数据的分析环节
    1.首先，数据库中的数据分析模型的选取，数据分析结果的导入

    2.从海量的数据中选取具有代表性的图片、产品，合成有针对性的趋势预测内容进 行发布

    3.选择报告中的关联信息，包括面料、辅料等信息，利用云计算的智能识别和推荐功能找到大量的相似产品以供上下游的使用

    4.生成一份完整的包含时效性和针对性的流行趋势预测报告

  3. 数据的成果与反馈环节
    数据的成果主要针对不同的受众会有不同的侧重部分，这也是大数据给予报 告制作者灵活变通的可能
